## ⚡ QUICK START (2 Minutes)

### Step 1: Start Server
```bash
cd gemini-ai-chat
python app.py
```

### Step 2: Open Browser
- **Chat:** `http://localhost:5000/dashboard`
- **Admin:** `http://localhost:5000/admin/dashboard`  
- **IDE:** `http://localhost:5000/ide`

### Step 3: Login
- Use Google OAuth or Email/Password
- Chat will auto-load your previous conversations
- Everything is stored permanently now!

### Step 4: Try These Features

**Chat Experience:**
```
1. Send message: "Write a Python web scraper"
2. Ask anything (code, ML, AWS, general knowledge)
3. All stored permanently in database
4. Previous chats auto-load on next login
```

**Admin Dashboard:**
```
1. Profile (top-right) → "Admin Dashboard"
2. See all your activity logs
3. Statistics & monitoring
4. Time-stamped audit trail
```

**Code IDE:**
```
1. Profile → "Code Editor"
2. Write any code
3. Click "Analyze with Sentinel AI"
4. Get instant code analysis
```

**Test Offline Protection:**
```
1. While chatting, press Ctrl+C to stop server
2. Chat will freeze
3. Refresh page → "Server unreachable" error (CORRECT!)
4. Restart server: python app.py
5. Everything works again
```
